
This repository contains my ongoing implementation of a probability-to-phase oracle construction for Hamiltonian-based quantum optimization.

The repository is intentionally presented in its current research state and contains implementation notebooks, verification code, references and intermediate experiments.

## Repository Contents

- `oracle_creation_*.ipynb` – Main implementation of the probability-to-phase oracle construction.
- `Formulas.ipynb` – Compact mathematical reference used throughout the implementation.
- `running_notes_for_phase_oracle_construction.pdf` – Handwritten research notes documenting intermediate derivations, ideas, and implementation planning.
- Other notebooks – Earlier implementations, tests and intermediate experiments retained as part of the process.
